ANTRAGO web konfiguration

Um den Konfigurationsprozess zu starten, wählen Sie bitte einer der zur Verfügung stehenden Optionen in der Seitennavigation aus.

Von dort aus, werden Sie aufgefordert die Konfigurationsdatei aus ihrem ANTRAGO webkomponenten-Projekt einzulesen. Sie finden Ihre Datei im jeweiligen Stammverzeichnis Ihres Frontends. Die Konfigurationsdatei ist eine JSON-Datei, die alle Einstellungen für die ANTRAGO webkomponenten enthält.

Wichtig: Dieses Programm macht kein selbstständiges Backup der Konfigurationsdatei. Beachten Sie dringend, vorher ein Backup Ihrer Konfiguration anzulegen. Nutzung auf eigene Verantwortung.

Bei Problemen, melden Sie sich bitte unter support@antrago.de oder jan.vonsondern@antrago.de.
